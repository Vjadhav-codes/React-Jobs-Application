Installation & Setup

Clone the repository:

git clone https://github.com/Vjadhav-codes/React-Jobs-Application.git
cd React-Jobs-Application
npm install


Start JSON Server:

npx json-server --watch data.json --port 5000


Run development server:

npm run dev
